Change default browser in Jupyter Notebook

Last Updated: Mar 29, 2018 10:34AM CDT

In order to change the default browser for Jupyter Notebook, first you need to generate the jupyter_notebook_config.py file by running the following command. You may already have this file, if so you can skip this step.

$ jupyter notebook --generate-config

You can now edit the c.NotebookApp.browser line in the config file found at ~/.jupyter/jupyter_notebook_config.py (OS X/ Linux) or C:\Users\builder\.juptyer\jupyter_notebook_config.py (Windows). The default line will look like this:

# c.NotebookApp.browser =''

Edit the line to point to your desired browser's executable. Take care to uncomment the line by deleting the '#'. In this example we will point to Google Chrome. 

MacOS:

c.NotebookApp.browser = u'open -a /Applications/Google\ Chrome.app %s'


Windows:
c.NotebookApp.browser = u'C:/Home/AppData/Local/Google/Chrome/Application/chrome.exe %s'


Optional Windows Configuration:

On some Windows setups the above option may not launch the desired browser and just generate a URL. You can use the following configuration instead.
 
import webbrowser
webbrowser.register('chrome', None, webbrowser.GenericBrowser('C:\Program Files (x86)\Google\Chrome\Application\chrome.exe'))
c.NotebookApp.browser = 'chrome'
 

After saving the change to your config file you can launch Jupyter Notebook using:

$ jupyter notebook
